# Minecraft Bedrock Add-on: Concept & Overview

## Add-on Information
- **Name**: Ore Add-on
- **Description**: Add new ore blocks, items (e.g. Budoh(item)/Budoh Ore(block))
- **Target Version**: 1.21+
- **Platform**: Bedrock
- **Language**: Japanese
- **AddOn Identifier**: `x051_ore_addon`

## Core Features
### Budoh Ore
- Add 'Budoh' item
    - Textrue: perple color like diamond
- Add 'Budoh Ore' block
    - Textrue: perple color like diamond ore
    - generate at deepslate layer
    - drop 'Budoh' items when break by pickaxe
- Add 'Budoh Block' block
    - Textrue: perple color like diamond block
    - crafted from 9 'Budoh' item by craft table
- Add 'Budoh Sword' item
    - Textrue: perple color like diamond sword
    - crafted from 2 'Budoh' items and 1 stick by craft table
    - Attack Damage: 10
- Add 'Budoh Helmet' item
    - Textrue: perple color like diamond helmet
    - crafted from 5 'Budoh' items by craft table
    - Defense: 3
- Add 'Budoh Chestplate' item
    - Textrue: perple color like diamond chestplate
    - crafted from 8 'Budoh' items by craft table
    - Defense: 8
- Add 'Budoh Leggings' item
    - Textrue: perple color like diamond leggings
    - crafted from 7 'Budoh' items by craft table
    - Defense: 6
- Add 'Budoh Boots' item
    - Textrue: perple color like diamond boots
    - crafted from 4 'Budoh' items by craft table
    - Defense: 3
